第一行的#!/bin/bash指的是默认采用的解释器
通常执行shell脚本有两种方式

  1. ./shellscript.sh (你必须保证脚本有可执行权限)
  2. /bin/sh shellscript.sh (这种方式直接执行了采用哪种解释器,故脚本第一行解释器可省略)
